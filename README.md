• 직관적인 Tableau 인터페이스와 인터랙티브 시각화를 설계하여 인력 배치 최적화 및 물량에 따른 트럭 및 인력 수요 예측에 기여함.

• 가상 시나리오 기반 분석 결과, 물류비용 절감과 인력운영의 효율성 향상을 예상함.
